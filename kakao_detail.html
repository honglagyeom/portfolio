kakao value

카카오의 문화와 지향하는 바를 나타내고싶어 기획한 섹션입니다.
사용자의 이목이 집중될 수 있게 카카오만의 감성이 담긴 이미지를 포함한 세부내용으로 디자인했으며, 숫자 인디케이터를 누르면 추가로 더 볼 수 있도록 제이쿼리 애니메이션작업을 했습니다.

code

kakao benefits

재생 버튼을 클릭하면 실시간으로 카카오ai동영상을 볼 수 있게 만든 섹션입니다. 또한 카카오만의 장점을 인터벌을 통해 일정한 간격을 두고 볼 수 있게 만들었으며, css애니메이션과 제이쿼리를 이용해 텍스트에 모션효과를 주었습니다. 화살표 버튼을 누르면 더 자세히 볼 수 있도록 작업했습니다.

code

kakao worth

브랜드의 가치를 나타내는 부분을 두가지 형태로 디자인하였습니다. 이미지와 같이가치 타이틀부분에 css hover를 이용해 간단한 모션효과를 주었습니다.

code

question/footer

question부분에 aos플러그인을 이용해 모션효과를 주었고, 배경색을 다르게해 눈에 띄게 디자인하였습니다. footer부분 텍스트에 커서를 올리면 강조될 수 있게 hover효과를 넣어줌으로써 마지막까지 카카오만의 감성을 담으려고 노력했습니다. 또한 back to top버튼을 누르면 상단으로 이동함으로써 사용자가 좀 더 편리하게 접근할 수 있도록 작업했습니다.
